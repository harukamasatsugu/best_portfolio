# best_portfolio
# 📈 日本株5銘柄 ポートフォリオ最適化分析

## 概要
本プロジェクトは、現代ポートフォリオ理論（MPT）に基づき、日本の主要5銘柄（トヨタ、ソニー、ソフトバンクG、三菱UFJ、NTT）の過去データを用いて、**モンテカルロ・シミュレーション**により**効率的フロンティア**を算出・プロットし、**最大シャープレシオ・ポートフォリオ**を特定します。

## 📊 使用銘柄
- トヨタ (7203.T)
- ソニー (6758.T)
- ソフトバンクG (9984.T)
- 三菱UFJ (8306.T)
- NTT (9432.T)
